## 歡迎任何人貢獻插件！

這是這個平台**最開放、最有價值**的地方：

> **任何人（不是只有我們）都可以設計插件，讓全香港的童軍區和旅團使用。**

### 為什麼要開放？

因為每個單位需求都不一樣。  
官方不可能做完所有東西。  
你的一個小工具，可能剛好解決很多人的問題。

### 門檻其實很低

- **第 2 級插件**（最推薦新手）：
  - 你只要部署**一份**系統
  - 所有區、所有旅團都能直接用
  - 主要只要讀 `?u=` 就能知道是哪個單位
  - 很多實用工具其實不需要很複雜

- **第 3 級插件**：
  - 你負責共用前端
  - 各單位自己接後端

### 怎麼開始貢獻？

1. 看 `docs/COMPONENT_SPEC.md`（特別是參數傳遞和 iframe 規則）
2. 做出一個簡單的插件（上面有最小骨架）
3. 決定好等級後，聯絡轉駁器維護者把插件登記進 registry

登記完成後，所有使用這套系統的單位，**馬上就能在「外掛市集」看到並安裝你的插件**。

### 我們很需要這些東西

活動報名、點名系統、圖書庫、營地預訂、團購、財務、進度追蹤…  
任何你覺得童軍會用到的工具都可以。

## 這是一個開放的插件市場（上架主要是幫忙登記）

我們的核心理念是：

**讓未來任何人寫插件，**  
**讓旅團和區覺得好用，**  
**就自己把插件插回他們的主系統，**  
**令他們的主系統越來越貼合他們真正的需要。**

### 實際上架方式

- **開發完全開放**：任何人（不管是不是官方）都可以自由開發插件。
- **要被大家看到，需要登記**：想讓區和旅團在「元件市場」看到你的插件，需要把插件資訊提供給這個轉駁器登記。
- **主要是幫忙登記**：我原則上是不會卡的，只要基本資訊齊全，會協助把插件登記上架。
- **只有設計有漏洞才會暫時不上架**：除非他們在設計上有漏洞，才有可能暫時不上架。

簡單來說：
> 寫插件誰都可以，**上架主要是幫忙登記**，我原則上是不會卡的，除非他們在設計上有漏洞，才有可能暫時不上架。
